🚀 COMANDOS DO DIA-A-DIA
═════════════════════════════════════════════════════════════════════════

🔄 COMEÇAR NOVO DIA

git checkout develop                    # Ir para develop
git pull                                # Trazer mudanças remoto
git checkout -b feature/sua-nova        # Criar nova feature


✏️ DURANTE O DIA

git status                              # Ver mudanças
git add .                               # Adicionar arquivos
git commit -m "✨ Feat: Descrição"     # Commit com emoji
git push                                # Enviar (primeira vez: -u origin)


☑️ FINALIZAR FEATURE

git push                                # Enviar último push
# Abrir GitHub/GitLab e criar Pull Request
# feature/sua-nova → develop


🔄 TRAZER MUDANÇAS DE DEVELOP

git fetch                               # Trazer do remoto
git rebase origin/develop              # Atualizar com develop


🧹 LIMPEZA

git branch -d feature/pronta            # Deletar local
git push origin --delete feature/pronta # Deletar remoto

🆘 TROUBLESHOOTING RÁPIDO
═════════════════════════════════════════════════════════════════════════

❌ "fatal: 'origin' does not exist"
✅ git remote add origin URL

❌ "Permission denied (publickey)"
✅ Configurar SSH ou usar HTTPS com token

❌ Cometi em develop/main!
✅ git reset --soft HEAD~1
   git checkout -b feature/correcao
   git add . && git commit -m "..."

❌ Preciso das mudanças mais recentes de develop
✅ git fetch && git rebase origin/develop

❌ Merge conflict?
✅ Resolver no editor, depois: git add . && git commit

❌ Quero descartar tudo
✅ git reset --hard HEAD
